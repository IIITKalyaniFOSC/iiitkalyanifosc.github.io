Introduction

IIIT Kalyani since long had a culture that encouraged students to take up competitive coding, but there is a another side of the coin. While competitive coding helps significantly in improving one's ability to implement complex logic in code and deepens the understanding of data structures and algorithms. It almost always after a certain level tends to move away from practicality. The work that a professional software engineer/developer does requires additional skills - learning how to contribute to open source projects, latest web technologies, modern mobile app development, robotics, operating systems, database management systems etc. are almost as essential as having a sound background in data structures and algorithms.

Goal

IIIT Kalyani Free and Open Source Community is here to bridge this gap, to create a culture that puts equal emphasis on approach to software development and the underlying implementation/algorithmic details. We regularly organise events to encourage students and promote open source softwares.
